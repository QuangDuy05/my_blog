Lập trình mạng với Java Socket: Xây dựng ứng dụng Chat đơn giản

Thiều Quang Duy December 29, 2025

Giới thiệu về Java Socket và Lập trình mạng

Lập trình mạng là một trong những kiến thức cốt lõi cho các kỹ sư phần mềm, đặc biệt là khi xây dựng các ứng dụng phân tán. Trong Java, Socket là một API mạnh mẽ giúp chúng ta thiết lập kết nối và truyền dữ liệu qua mạng một cách dễ dàng. Bài viết này sẽ hướng dẫn bạn cách tạo một ứng dụng chat đơn giản sử dụng Java Socket.

Socket là gì?

Socket là một endpoint của một liên kết truyền thông hai chiều giữa hai chương trình chạy trên mạng. Về cơ bản, nó là một "điểm cuối" cho phép các ứng dụng gửi và nhận dữ liệu qua một kết nối mạng.

Mô hình Client-Server sử dụng Java Socket

Các bước xây dựng ứng dụng Chat

1. Server-side (Máy chủ)

Máy chủ sẽ lắng nghe các kết nối đến từ phía Client. Khi có Client kết nối, Server sẽ tạo một Socket mới để giao tiếp với Client đó.

import java.io.*;
import java.net.*;

public class ChatServer {
    public static void main(String[] args) throws IOException {
        int port = 12345;
        ServerSocket serverSocket = new ServerSocket(port);
        System.out.println("Server đang lắng nghe trên cổng " + port);

        while (true) {
            Socket clientSocket = serverSocket.accept();
            System.out.println("Client mới đã kết nối: " + clientSocket);

            new ServerThread(clientSocket).start();
        }
    }
}

class ServerThread extends Thread {
    private Socket clientSocket;
    private PrintWriter out;
    private BufferedReader in;

    public ServerThread(Socket socket) {
        this.clientSocket = socket;
    }

    public void run() {
        try {
            out = new PrintWriter(clientSocket.getOutputStream(), true);
            in = new BufferedReader(new InputStreamReader(clientSocket.getInputStream()));

            String inputLine;
            while ((inputLine = in.readLine()) != null) {
                System.out.println("Client " + clientSocket.getPort() + ": " + inputLine);
                out.println("Server nhận được: " + inputLine);
            }

            in.close();
            out.close();
            clientSocket.close();
        } catch (IOException e) {
            System.out.println("Lỗi: " + e.getMessage());
        }
    }
}

2. Client-side (Máy khách)

Phía Client sẽ tạo một Socket để kết nối đến địa chỉ IP và cổng của Server. Sau đó, Client có thể gửi và nhận tin nhắn.

import java.io.*;
import java.net.*;
import java.util.Scanner;

public class ChatClient {
    public static void main(String[] args) throws IOException {
        String hostName = "localhost";
        int port = 12345;

        try (Socket socket = new Socket(hostName, port);
             PrintWriter out = new PrintWriter(socket.getOutputStream(), true);
             BufferedReader in = new BufferedReader(new InputStreamReader(socket.getInputStream()));
             Scanner scanner = new Scanner(System.in)) {

            System.out.println("Đã kết nối tới Server. Nhập tin nhắn của bạn:");
            String userInput;
            while ((userInput = scanner.nextLine()) != null) {
                out.println(userInput);
                System.out.println("Server phản hồi: " + in.readLine());
                if (userInput.equals("bye")) break;
            }

        } catch (UnknownHostException e) {
            System.err.println("Không biết host " + hostName);
            System.exit(1);
        } catch (IOException e) {
            System.err.println("Không thể lấy I/O cho kết nối tới " + hostName);
            System.exit(1);
        }
    }
}

Kết luận

Với Java Socket, chúng ta có thể dễ dàng xây dựng các ứng dụng mạng cơ bản đến phức tạp. Hiểu rõ cách hoạt động của Socket là chìa khóa để phát triển các hệ thống phân tán mạnh mẽ và hiệu quả. Hy vọng qua bài viết này, bạn đã có cái nhìn tổng quan và có thể tự tay xây dựng ứng dụng chat đầu tiên của mình!
