Bảo mật kết nối Socket với SSL/TLS

Khi xây dựng ứng dụng Chat hay hệ thống quản lý bãi xe, dữ liệu cá nhân của người dùng cần được bảo vệ. Việc gửi dữ liệu qua Socket thông thường (Plaintext) có thể bị hacker đánh cắp dễ dàng qua kỹ thuật Sniffing.

Dưới đây là mã nguồn triển khai một SSLServerSocket trong Java, giúp mã hóa toàn bộ thông tin truyền tải giữa Client và Server.

import javax.net.ssl.*;
import java.io.*;

public class SecureServer {
    public static void main(String[] args) throws Exception {
        // Thiết lập chứng chỉ bảo mật cho Server
        System.setProperty("javax.net.ssl.keyStore", "mykeystore.jks");
        System.setProperty("javax.net.ssl.keyStorePassword", "duy123");

        SSLServerSocketFactory ssf = (SSLServerSocketFactory) SSLServerSocketFactory.getDefault();
        SSLServerSocket serverSocket = (SSLServerSocket) ssf.createServerSocket(8888);

        System.out.println("Server bảo mật đang chờ kết nối...");
        try (SSLSocket socket = (SSLSocket) serverSocket.accept()) {
            PrintWriter out = new PrintWriter(socket.getOutputStream(), true);
            out.println("Dữ liệu của bạn đã được mã hóa an toàn!");
        }
    }
}
