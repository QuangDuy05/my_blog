BÀI 1: TỔNG QUAN VỀ JAVA VÀ CÀI ĐẶT MÔI TRƯỜNG

Thiều Quang Duy
December 28, 2025
Updated on December 28, 2025

 

  • Nội dung: Java là một trong những ngôn ngữ lập trình phổ biến nhất thế giới, được phát triển bởi Sun Microsystems (nay thuộc Oracle). Với triết lý "Write Once, Run Anywhere", mã nguồn Java sau khi biên dịch thành bytecode có thể chạy trên bất kỳ thiết bị nào có cài đặt Java Virtual Machine (JVM). Đối với sinh viên HUTECH, Java là nền tảng quan trọng để tiếp cận lập trình hướng đối tượng (OOP).

    Các bước thực thi một chương trình Java:

    1. Viết mã nguồn trong file .java.

    2. Biên dịch bằng javac thành file .class (bytecode).

    3. Chạy chương trình thông qua JVM.

    Code mẫu Hello World:

  • public class HelloWorld {
    public static void main(String[] args) {
        // In ra màn hình lời chào mừng
        System.out.println("Chào mừng thầy đến với Blog của Duy HUTECH!");
        System.out.println("Phiên bản Java đang sử dụng: 17 LTS");
    }
}
