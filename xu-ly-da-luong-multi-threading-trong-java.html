Xử lý Đa luồng (Multi-threading) trong Java

Thiều Quang Duy

Tại sao cần Đa luồng trong quản lý bãi xe?

Hãy tưởng tượng bãi giữ xe của bạn có hàng trăm xe ra vào cùng lúc. Nếu chỉ xử lý đơn luồng, xe này phải đợi xe kia thanh toán xong mới được vào. Đa luồng (Multi-threading) cho phép hệ thống xử lý nhiều yêu cầu đồng thời, giúp camera nhận diện biển số và máy in vé hoạt động cùng lúc.

Ví dụ code mô phỏng luồng xe:

class ParkingGate extends Thread {
    private String gateName;
    public ParkingGate(String name) { this.gateName = name; }
    public void run() {
        System.out.println(gateName + " đang xử lý xe...");
    }
}
