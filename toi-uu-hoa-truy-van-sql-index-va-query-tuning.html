Tối ưu hóa truy vấn SQL - Index và Query Tuning

Khi database lớn (hàng triệu bản ghi), truy vấn chậm là vấn đề lớn. Index và tối ưu hóa query giúp tăng tốc đáng kể.

1. Tạo Index cho cột thường xuyên tìm kiếm

CREATE INDEX idx_bien_so ON Xe(bien_so);
CREATE INDEX idx_thoi_gian_vao ON Ve(thoi_gian_vao);

2. Sử dụng EXPLAIN để phân tích query

EXPLAIN SELECT * FROM Xe WHERE bien_so = '59A1-12345';

Áp dụng index đúng cách có thể tăng tốc query từ vài giây xuống mili giây!
