BÀI 2: HỆ THỐNG CÁC KIỂU DỮ LIỆU TRONG JAVA

Thiều Quang Duy
December 28, 2025
Updated on December 28, 2025

Nội dung: Trong Java, kiểu dữ liệu được chia làm hai loại chính: Kiểu nguyên thủy (Primitive) và Kiểu tham chiếu (Reference). Việc chọn đúng kiểu dữ liệu giúp chương trình chạy nhanh hơn và tiết kiệm bộ nhớ RAM.

  • Kiểu số nguyên: byte, short, int, long.

  • Kiểu số thực: float, double.

  • Kiểu ký tự và logic: char, boolean.

  • Kiểu tham chiếu: String, Array, Object.

Code mẫu minh họa:

public class DataStyles {
    public static void main(String[] args) {
        String studentName = "Thiều Quang Duy";
        int age = 20;
        double mathScore = 8.5;
        boolean isPassed = true;

        System.out.println("Thông tin sinh viên: " + studentName);
        System.out.println("Điểm toán: " + mathScore + " - Trạng thái: " + (isPassed ? "Đạt" : "Trượt"));
    }
}
