Kết nối Cơ sở dữ liệu (JDBC)

Thiều Quang Duy December 29, 2025

Nền tảng của một Data Engineer

Việc lấy dữ liệu từ Database là kỹ năng tối quan trọng. JDBC (Java Database Connectivity) là cầu nối giúp ứng dụng Java nói chuyện với SQL Server hoặc MySQL.

Ní cần chuẩn bị Driver JDBC và chuỗi kết nối (Connection String) để bắt đầu truy vấn dữ liệu.
