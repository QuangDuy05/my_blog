BÀI 4: VÒNG LẶP FOR, WHILE VÀ DO-WHILE

Thiều Quang Duy
December 28, 2025
Updated on December 28, 2025

Nội dung: Vòng lặp là giải pháp để thực hiện các công việc lặp đi lặp lại. for thường dùng khi biết trước số lần lặp, while dùng khi lặp dựa trên một điều kiện còn đúng, và do-while đảm bảo khối lệnh được thực hiện ít nhất một lần trước khi kiểm tra điều kiện.

Code mẫu tính tổng số chẵn:

public class LoopExample {
    public static void main(String[] args) {
        int sum = 0;
        for (int i = 1; i <= 100; i++) {
            if (i % 2 == 0) {
                sum += i;
            }
        }
        System.out.println("Tổng các số chẵn từ 1 đến 100 là: " + sum);
    }
}
