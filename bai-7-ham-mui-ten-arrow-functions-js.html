BÀI 7: HÀM MŨI TÊN (ARROW FUNCTIONS) TRONG ES6

Thiều Quang Duy
December 28, 2025
Updated on December 28, 2025

Nội dung: Arrow functions mang lại cú pháp ngắn gọn hơn cho việc viết hàm trong JavaScript. Ngoài việc bớt chữ function, nó còn xử lý từ khóa this một cách thông minh hơn trong các ứng dụng web phức tạp.

Code mẫu:

// Hàm tính diện tích hình chữ nhật
const calculateArea = (width, height) => width * height;

console.log("Diện tích: " + calculateArea(5, 10));

// Ví dụ hàm không tham số
const sayHello = () => console.log("Chào bạn đã đến với Blog của Duy!");
sayHello();
