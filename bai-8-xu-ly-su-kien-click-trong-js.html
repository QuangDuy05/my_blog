BÀI 8: XỬ LÝ SỰ KIỆN DOM TRONG JAVASCRIPT

Thiều Quang Duy
December 28, 2025
Updated on December 28, 2025

Nội dung: DOM (Document Object Model) cho phép JavaScript thay đổi nội dung, cấu trúc và định dạng của trang web. Xử lý sự kiện (Event Handling) là cách web phản hồi lại hành động của người dùng như Click, Hover, hoặc Submit form.

Code mẫu thay đổi nội dung trang web:

// HTML: <button id="btn">Click me</button> <h1 id="title">Chào HUTECH</h1>

const button = document.getElementById('btn');
const title = document.getElementById('title');

button.addEventListener('click', () => {
    title.innerText = "Bạn đã bấm nút thành công!";
    title.style.color = "blue";
});
