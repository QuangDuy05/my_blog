BÀI 6: JAVASCRIPT HIỆN ĐẠI - PHÂN BIỆT VAR, LET VÀ CONST

Thiều Quang Duy
December 28, 2025
Updated on December 28, 2025

Nội dung: Trước ES6, var là cách duy nhất để khai báo biến nhưng nó gây ra nhiều lỗi do tính chất hoisting. Ngày nay, let và const được khuyến khích sử dụng:

  • const: Dùng cho các giá trị không đổi (hằng số).

  • let: Dùng cho biến có thể thay đổi giá trị và có phạm vi trong khối lệnh { } (block scope).

Code mẫu:

const PI = 3.14159;
let radius = 10;

if (radius > 0) {
    let area = PI * radius * radius;
    console.log("Diện tích hình tròn là: " + area);
}
// console.log(area); // Lỗi! area không tồn tại ngoài block if
