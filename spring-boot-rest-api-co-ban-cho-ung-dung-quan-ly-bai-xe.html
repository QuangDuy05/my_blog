Spring Boot REST API cơ bản cho ứng dụng quản lý bãi xe

Thiều Quang Duy

Spring Boot giúp tạo REST API nhanh chóng. Ví dụ: API quản lý chỗ trống bãi xe.

@RestController
@RequestMapping("/api/parking")
public class ParkingController {
    @Autowired
    private ParkingService service;

    @GetMapping("/slots")
    public ResponseEntity getAvailableSlots() {
        return ResponseEntity.ok(service.getAvailableSlots());
    }
}
