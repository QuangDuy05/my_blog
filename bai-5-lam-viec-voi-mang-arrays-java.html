BÀI 5: QUẢN LÝ DỮ LIỆU VỚI MẢNG (ARRAY)

Thiều Quang Duy
December 28, 2025
Updated on December 28, 2025

Nội dung: Mảng là một tập hợp các phần tử có cùng kiểu dữ liệu được lưu trữ tại các vị trí bộ nhớ liên tiếp. Trong Java, mảng là một đối tượng và có kích thước cố định sau khi khởi tạo. Chỉ số (index) của mảng luôn bắt đầu từ 0.

Code mẫu tìm giá trị lớn nhất trong mảng:

public class ArrayDemo {
    public static void main(String[] args) {
        int[] scores = {7, 9, 5, 10, 8};
        int max = scores[0];
        
        for (int i = 1; i < scores.length; i++) {
            if (scores[i] > max) {
                max = scores[i];
            }
        }
        System.out.println("Điểm cao nhất trong danh sách là: " + max);
    }
}
