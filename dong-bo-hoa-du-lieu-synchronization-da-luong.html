Đồng bộ hóa dữ liệu (Synchronization) đa luồng

Thiều Quang Duy December 29, 2025

Xử lý đồng bộ trong Hệ thống quản lý bãi xe

Trong bãi giữ xe, nếu hai camera cùng nhận diện hai xe vào một lúc và cùng cập nhật số chỗ trống, sẽ xảy ra lỗi Race Condition. Điều này khiến số lượng chỗ trống hiển thị bị sai lệch.

Giải pháp là sử dụng ReentrantLock để đảm bảo tại một thời điểm chỉ có một tiến trình được phép thay đổi biến availableSlots.

import java.util.concurrent.locks.ReentrantLock;

public class ParkingLotManager {
    private int slots = 100;
    private final ReentrantLock lock = new ReentrantLock();

    public void carEntry() {
        lock.lock(); // Bắt đầu khóa tài nguyên
        try {
            if (slots > 0) {
                slots--;
                System.out.println("Xe đã vào. Chỗ trống còn: " + slots);
            }
        } finally {
            lock.unlock(); // Luôn mở khóa để xe khác vào
        }
    }
}
