Xử lý dữ liệu lớn (Big Data) trong Java

Trong vai trò Data Engineer, việc đọc các file dữ liệu hàng trăm MB hay hàng GB là chuyện thường ngày. Nếu sử dụng các phương pháp đọc file thông thường, hệ thống sẽ rất dễ bị treo do tràn bộ nhớ (Heap Space).

Để tối ưu, chúng ta sử dụng BufferedReader kết hợp với Stream API. Cách tiếp cận này giúp ứng dụng xử lý dữ liệu theo từng dòng, giảm tải cho RAM đáng kể.

import java.io.*;
import java.nio.file.*;

public class DataProcessor {
    public static void main(String[] args) {
        Path path = Paths.get("parking_history_big_data.csv");
        // Kỹ thuật xử lý luồng để tiết kiệm tài nguyên
        try (BufferedReader reader = Files.newBufferedReader(path)) {
            String line;
            long count = 0;
            while ((line = reader.readLine()) != null) {
                count++;
                // Giả lập xử lý dữ liệu (ETL process)
                String[] data = line.split(",");
                if (count % 10000 == 0) {
                    System.out.println("Đã nạp thành công: " + count + " dòng");
                }
            }
            System.out.println("Hoàn tất xử lý tập tin!");
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}
